# SmallTalk
### English
A small Discord Chatbot

You can talk with him, for example when you have no friends :/

Just go to a discord server and begin to talk with him!

#### The topics he can talk about:
- weather
- greeting
- question
- jokes
- age
- doing
- wishes
- boredom
- thanks
- refusal
- happy
- bot
- opinions
- funny
- colors
- mood
- agree
- leaving
- insults
- amazed
